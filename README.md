htlp
============================

## Requirements
- yarn 1.13.0
- firebase-tools 6.3.0

## First
- `yarn`

```
brew install yarn
```

- `firebase-tools`

```
yarn global add firebase-tools
```


## Devlopment
- `http://localhost:5000`

```
firebase serve
```

## Deploy
- Hosting to Firebase

```
firebase deploy
```
