htlp
============================

## Requirements
- firebase-tools 6.3.0

## First
- Install `yarn`

```
brew install yarn
```

- Install `firebase-tools`

```
yarn global add firebase-tools
```


## Devlopment
- Access to `http://localhost:5000`

```
firebase serve
```

## Deploy
- Hosting to Firebase

```
firebase deploy
```
